# Knowledge Base

The knowledge-base contains reusable engineering knowledge only.

## Scope

- `10-engineering-principles/` contains stable cross-project reasoning rules.
- `20-coding-standards/10-core/` contains core technical standards.
- `20-coding-standards/20-language/` contains language-specific standards.
- `20-coding-standards/30-application-type/` contains standards for API, mobile,
  console, worker, and other application types.
- `20-coding-standards/40-framework/` contains framework-specific standards.

## Precedence

Within the KB, higher-precedence knowledge appears earlier:

1. `10-engineering-principles`
2. `20-coding-standards/10-core`
3. `20-coding-standards/20-language`
4. `20-coding-standards/30-application-type`
5. `20-coding-standards/40-framework`

Lower-precedence KB layers may refine earlier layers, but may not contradict or
weaken them.
